Navigated to:

https://www.tn.gov/health/cedep/ncov/data/downloadable-datasets.html

downloaded the Daily Case Information data set on COVID-19 cases in State of Tennessee (Excel spreadsheet attached).

At Matlab command line:

D=[];

Then double click on "D" variable in the Workspace window to open up a (blank) Matlab spreadsheet for that variable.

Copied column P from the Tennessee data spreadsheet, skipping the first row.

Pasted the data into Matlab variable D using the "Paste Excel Data" function after right clicking on the first element.

Then, from the command line:

plot(D)
D=D(906:-1:1);
plot(D)
grid
t=[1:906];
A=[ones(906,1),t'];
c=inv(A'*A)*A'*D
plot(t,D,'o',t,c(1)+c(2)*t)
grid

These command accomplished the following:

  1. plotted Total Deaths data, noticed that it is reverse chronological (decreasing deaths)
  2. reversed the order of the elements in D
  3. added a grid to graph
  4. created an integer variable "t" indicating the days since March 4, 2020
  5. built A, the matrix of modeling functions
  6. solved for the best fit model parameters in c, corresponding to simple line
  7. plotted the data on the same axes as the best fit model
  8. added a grid
